# Система распознавания автомобильных номеров 

**Веб-приложение для автоматического детектирования и распознавания автомобильных номеров на изображениях с использованием современных методов компьютерного зрения и глубокого обучения.**

---

##  Основные возможности

- **Загрузка изображений** через удобный веб-интерфейс
- **Автоматическое детектирование** номерных знаков с помощью YOLOv11n
- **Распознавание текста** с использованием трансформерной модели TrOCR
- **Визуализация результатов** с bounding boxes и распознанным текстом
- **Статистика обработки** и подробная информация по каждому найденному номеру
- **Система логирования** всех операций распознавания

---

##  Технологический стек

### Backend
- **FastAPI** — асинхронный веб-фреймворк с автоматической документацией OpenAPI
- **Ultralytics YOLO** — детекция объектов (YOLOv11n)
- **Hugging Face Transformers** — распознавание текста (TrOCR Large Printed)
- **PyTorch** — фреймворк глубокого обучения
- **OpenCV** — обработка и визуализация изображений

### Frontend
- **Jinja2** — шаблонизация HTML
- **Чистый CSS/JavaScript** — адаптивный интерфейс
- **Bootstrap** элементы для стилизации
